Python Automation for IP Allow List Management

Overview:

This project demonstrates the use of Python automation to manage an IP allow list used to control access to a restricted healthcare subnetwork containing sensitive patient information. The script automates the process of identifying and removing unauthorized IP addresses from an approved access list based on a separate removal list.

Objectives:

  • Automate security-related administrative tasks using Python
  • Improve access control management efficiency
  • Reduce manual errors during allow list maintenance
  • Strengthen understanding of file handling and data parsing in Python

Tools & Technologies:

  • Python
  • Jupyter Notebook
  • Text Files (.txt)
  • File Handling Functions
  • Lists & Loops

Key Activities:

Opened and Read Security Allow List Files

  • Used Python file handling functions to open and read access control files
  • Retrieved IP address data from allow and remove lists
Parsed and Converted Data for Processing
  • Converted file contents into Python lists for easier processing
  • Organized IP address data for comparison and filtering
Automated IP Address Removal
  • Iterated through the remove list using loops and conditional logic
  • Removed unauthorized IP addresses from the allow list automatically
Updated Access Control File
  • Wrote the revised list of approved IP addresses back into the file
  • Verified that unauthorized entries were successfully removed

Security Relevance:

This project demonstrates foundational cybersecurity concepts involving:

  • Access control management
  • Security automation
  • Identity and network access restrictions
  • Administrative scripting
  • Data integrity and accuracy

Scenario:

This scenario is based on a fictional company:

You are a security professional working at a health care company. As part of your job, you're required to regularly update a file that identifies the employees who can access restricted content. The contents of the file are based on who is working with personal patient records. Employees are restricted access based on their IP address. There is an allow list for IP addresses permitted to sign into the restricted subnetwork. There's also a remove list that identifies which employees you must remove from this allow list.

Your task is to create an algorithm that uses Python code to check whether the allow list contains any IP addresses identified on the remove list. If so, you should remove those IP addresses from the file containing the allow list.
